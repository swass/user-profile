ngrx-learning works!

  1. install ngrx store in app : ng add @ngrx/store
  2. package.json and app module will be updated, store will be imported in app module
  3. define initial state, and reducers
  4. define actions to be dispatched from component
  5. inject Store object in component constructor
  6. using store object, subscribe to the app states or dispach any action from comp.
